Step 1
CSV 내보내기
경기 결과와 순위를 CSV로 복사합니다. Google Sheets에서 A1 선택 후 Ctrl+V 붙여넣기하세요.
Step 2
Google Sheets 실시간 연동
Apps Script Web App URL 입력 시 버튼 하나로 자동 동기화됩니다.
① Google Sheets 열기 → 상단 확장 프로그램 → Apps Script
② 편집기 코드 전부 삭제 → 아래 코드 붙여넣기 → Ctrl+S 저장
③ 상단 [배포] → [새 배포] → 유형: 웹 앱 선택
④ 실행: 나 / 액세스 권한: 모든 사용자 → [배포]
⑤ 생성된 웹 앱 URL 복사 → 위 입력란에 붙여넣기
